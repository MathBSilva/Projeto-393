# Classificador de Imagens K-NN

## Como executar o programa

**Passo 1:** Acesse a pasta através do link.
https://drive.google.com/drive/folders/1Sg2R6x7q88zVMD9y5_7ufIsEqKCqlvXf?usp=sharing
